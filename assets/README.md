# 头像素材

把你的虚拟形象 PNG 放到这里，文件名建议是 `avatar.png`。

如果你要做多头像切换，请把素材放到 `assets/avatars/`，例如：

- `assets/avatars/cyber_girl.png`
- `assets/avatars/anchor_blue.png`
- `assets/avatars/avatar_male.png`
- `assets/avatars/avatar_female.png`

如果你要用 GPIO0/GPIO1 进行四种虚拟形象选择，建议使用以下文件名：

- `assets/avatars/avatar_00.png`
- `assets/avatars/avatar_01.png`
- `assets/avatars/avatar_10.png`
- `assets/avatars/avatar_11.png`

要求：

1. 最好带透明通道。
2. 尽量使用正方形或接近正方形素材。
3. 头像越清晰，叠加效果越自然。
